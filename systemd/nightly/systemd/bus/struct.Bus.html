pub fn process(&mut self) -> Result<Option<Option<Message>>>[src]

Drives the connection between the client and the message bus. Each time it is invoked a single operation is executed.

Returns None if no operations were pending (and thus no operations were processed). Returns Some(None) if progress was made but no message was processed. Returns Some(Message) if a message was processed.

This corresponds to sd_bus_process.
