pub struct Utf8CStr { /* fields omitted */ }

A wrapper that promises it's contents are null-terminated & validly utf-8 encoded

Plain std::ffi::CStr only promises null termination, but some ffi (and other bits) require strings that are both valid utf8 and null terminated.
