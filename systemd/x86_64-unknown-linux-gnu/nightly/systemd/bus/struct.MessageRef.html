Set to true to allow the bus to launch an owner for the destination name.

Set to false to prevent the bus from launching an owner for the destination name.

Fails if the message is sealed

This controls the NO_AUTO_START dbus header flag.

The specification covers some details about the auto start mechanism, but not all of it is specified.

pub fn iter<'a>(&'a mut self) -> Result<MessageIter<'a>>[src]

Get an iterator over the message. This iterator really exists with in the Message itself, so we can only hand out one at a time.

Ideally, handing this iterator out wouldn't prevent the use of other non-iterator accessors, but right now it does (unless you bypass borrowck using unsafe{})

Requires that message is sealed.
