Module systemd::bus [−] [src]

An interface to work with the dbus message bus.

WARNING: this is not complete. Right now we're missing:

  • message encoding/decoding
  • server support
  • async client support

In short, the only functional thing is issuing blocking dbus calls with pre-populated messages and writing custom ffi decoders of the message replies.
